# Rick and Morty Wiki

This project was developed as a midterm assignment for the **Web Programming (SE 3355)** course.

The application is a small wiki that allows users to explore characters, episodes, and locations from the Rick and Morty universe using the public Rick and Morty API.

---

## Features

- Browse characters with pagination
- Search characters by name
- Filter characters by status (Alive, Dead, Unknown)
- View detailed character pages
- Browse episodes and episode details
- View characters appearing in an episode
- Browse locations with type and dimension information
- Responsive and custom UI built with Tailwind CSS

---

## Technologies Used

- React (Vite)
- React Router
- Tailwind CSS
- JavaScript (ES6+)
- Rick and Morty Public API

---
